# wlroots compositor written in V

Contributions are welcome, as well as suggestions on how to improve the [wlroots bindings](wlr).

# Building
This project depends on the following packages:
- `wlroots-0.20`
- `wayland`
- `xkbcommon`

# Supported protocols
- `wayland`
- `xdg-output`
- `xdg-shell`
- `wlr-layer-shell-unstable-v1`
